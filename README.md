# React: Session 2 - Digging deeper

This is a companion repository for the [React Presentation](https://github.com/danm0e/react-training-presentation).

### This exercise covers

- State colocation principles and best practices
- Performance optimisation with useCallback, React.memo, and useReducer

### Getting Started

Make sure you're on the a version of node >= 20!

To get started simply run

```bash
npm i
```

Followed by

```bash
npm start
```
